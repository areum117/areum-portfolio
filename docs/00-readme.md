# Portfolio Redesign Brief

## 목표

현재 포트폴리오를 단순한 개발자 이력서 페이지가 아니라,
“AI 기능을 서비스 흐름으로 연결하는 개발자”라는 정체성이 명확하게 드러나는
독창적인 웹 포트폴리오로 리디자인한다.

## 핵심 메시지

AI 기능을 단순히 구현하는 데서 끝내지 않고,
사용자 입력 → AI 서버 → Backend API → Frontend UI → 사용자 확인 → 데이터 저장까지
하나의 서비스 경험으로 연결하는 웹 개발자.

## 절대 피해야 할 것

- 흔한 Hero + About + Project + Stack + Contact 구조
- 단순 Bento Grid 카드 나열
- 흔한 다크모드 AI SaaS 대시보드
- 프로젝트 설명문이 긴 보고서형 구성
- 모든 카드가 같은 사각형 박스로 반복되는 디자인
- 몽글마을 이미지를 큰 배경 이미지로 사용하는 방식
- 기술 태그만 단순 나열하는 Stack 섹션

## 최종 인상

채용 담당자가 봤을 때 다음 인상이 들어야 한다.

“이 사람은 AI 기능을 웹 서비스 안에 실제로 연결해본 사람이다.”  
“단순히 화면을 만드는 게 아니라 서비스 흐름을 설계할 줄 안다.”  
“포트폴리오 자체도 자기 강점을 잘 보여주도록 설계했다.”
