## AI Service Flow

제목:

단계:
1. User Input
2. Context Check
3. AI Planning
4. API Bridge
5. UI Preview
6. Confirm & Save

연결선 위에는 message, plan_json, job_id, todo_items, user_confirm 같은 작은 데이터 라벨을 표시한다.
